Cascade

The cascade is a fundamental feature of CSS. It is an algorithm defining how to combine properties values originating from different sources. It lies at the core of CSS as stressed by its name: Cascading Style Sheets.

Example

If you apply a CSS style to a tag, the affects of that style will cascade to all elements nested within that tag:
 <body>
    <p>This paragraph will inheret the style from the body.<p>
</body>
body {
  background-color: green;
  color: white;
}
This will display on the page as:

This paragraph will inheret the style from the body.
